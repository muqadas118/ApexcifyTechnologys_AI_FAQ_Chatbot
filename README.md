# ApexcifyTechnologys_AI_FAQ_Chatbot

## 📌 Project Overview
This project is an AI FAQ Chatbot developed as part of the Apexcify Technologys Internship Program.

The chatbot answers Artificial Intelligence related questions using Natural Language Processing (NLP) techniques and provides an interactive GUI interface.

---

## 🚀 Features
- AI FAQ Question Answering
- NLP based similarity matching
- Interactive GUI using Gradio
- Confidence score for responses
- Beginner-friendly AI implementation

---

## 🧠 Technologies Used
- Python
- Scikit-learn
- NLP (TF-IDF Vectorization)
- Gradio Interface

---

## ▶️ How to Run

Install dependencies:
- pip install -r requirements.txt
- Run chatbot file
